Conway's Game of Life

Over Christmas break, my brother challenged me to a code-off. We coded up Conway's Game of Life cell simulation and compared results :). I used Python with tkinter for my GUI. This implementation is able to run the simulation, allows for selecting and deselecting cells for custom designs, and increasing or slowing down the speed of the simulation iteration.

Finite State Automota

This is an ongoing project. I am working on writing a simple scripting language that allows a user to define finite state machines. The parser will then be able to simulate the machine and check if input strings are accepted by the machine's language. My code currently can handle simulating programatically created deterministic machines, but I am working on extending the capability to full parsing of the scripting language and non-determinism. My source code for this project is currently not available for public view. Contact me if you would like to access it.

Research

SHA Algorithm Verification

For the last few years, I have been working in Dr. Eric Mercer's Verification and Validation lab at BYU. As part of that, I have worked on performing functional equivalence verification of OpenSSL's SHA-2 and SHA-3 algorithm families to the defined NIST standards.
The work done on SHA-2 was published in NASA's Formal Methods conference (NFM 2021). The contents of that work can be found here.
Our work on SHA-3 is currently being reviewed by the NFM 2022 committee.
