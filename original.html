Projects:

  • SHA Algorithm Verification. As part of Dr. Mercer's lab here at BYU, I have participated in several projects proving out functional equivalence of OpenSSL's SHA implementations to the NIST standards. We have done this for portions of the SHA2-256 and SHA3-256 algorithms. Our work on the SHA2 algorithm was accepted to the Nasa Formal Methods conference for 2021, and our SHA3 work is currently in review for the Nasa Formal Methods 2022 conference.
  • Conway's Game of Life. This is a small project I did over christmas break. I created a GUI application (using tkinter in python) that runs conways game of life, and allows for user interaction in selecting cells, and changing the refresh speed.
  • Finite State Machine Emulator. This is an ongoing project. I have been working on writing a simple scripting language to define finite state machines, and a program that can interpret the script, build a model of the machine, and run test inputs through that model. It is currently in development, but I have it working properly for deterministic instances.
